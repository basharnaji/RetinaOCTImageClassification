About the model

The model was trained on a limited amount of OCT images of four different classes (CNV - Choroidal Neovascularization, DME - Diabetic Macular Edema, DRUSEN and NORMAL). Data augmentation was performed using synthetic images that were generated using the SinGAN approach (Shaham et al., 2019).
